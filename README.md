### uwayss.com

This is the source code for my site. Right now, it's just a mirror of my GitHub README.

Want to use it? (why...) Just:

1. **Clone** the repo.
2. Run `pnpm install`.
3. Run `pnpm get-readme {your-github-username}`.
4. Build it with `pnpm build`.
5. **Done.** Just host the `dist/` folder wherever you want.
